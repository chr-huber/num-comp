{% blocktrans trimmed %} Your task in Stage 1 is to correctly solve as many addition exercises as possible. To be more precise, you will have 3 minutes’ time in order to solve as many additions of five randomly selected two-digit numbers as possible, by entering the sum of the five numbers. You are not allowed to use calculators. You enter an answer by clicking with the mouse on the "Confirm" button. When you enter an answer, you immediately find out on the screen whether it was correct or not. {% endblocktrans %}

{% trans "Payment" %}

{% blocktrans trimmed %} If Stage 1 is the stage selected for payment (in Part 1), then you will receive €0.50 (i.e., 50 cent) for each correct answer that you entered within the 3 minutes. Your payment is not reduced when you enter a wrong answer. From now on, we call this method of payment the Piece-rate payment. {% endblocktrans %}

{% trans "Trial round" %}

{% blocktrans trimmed %} Directly before the start of this stage you will be given one minute in order to familiarize yourselves with the screen: During this time you can solve addition exercises, which do not count for the experiment. Afterwards, Stage 1 will begin. {% endblocktrans %}
